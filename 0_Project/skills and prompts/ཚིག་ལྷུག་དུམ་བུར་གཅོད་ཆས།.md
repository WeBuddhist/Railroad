##### མི་དགེ་བ་མི་འཕེལ་བའི་ཕན་ཡོན།

#### ཚེ་ཕྱི་མ་ལ་བསམ་པའི་ཕན་ཡོན།

1.3 བཟོད་པ་སྒོམ་དགོས་པར་བསྟན་པ།

  

- གསར་དུ་བསྣན་པའི་མགོ་བརྗོད་རྣམས་ཀྱི་གོང་འོག་ཏུ་ཐིག་ཕྲེང་སྟོང་པ་རེ་རེ་ངེས་པར་དུ་འཇོག་དགོས། འདི་ནི་གཞུང་ཚིག་དངོས་དང་མི་ནོར་བའི་ཆེད་དུ་ཡིན།

  

  

2. གཞུང་ཚིག་རྣམས་དུམ་བུ་ (obsidian block) ལ་གཅོད་རོགས།

- ས་བཅད་རེ་རེ་ནང་གི་གཞུང་ཚིག་རྣམས་དོན་ཚན་ཐུང་ངུའི་དུ་བུར་བཅད་དགོས། (དུམ་བུ་རེ་རེ་ལ་ཚིག་བར་བཞི་བཅུ་ལས་བརྒལ་མི་ཆོག)

- ཚིག་བཅད་རྣམས་ཤློཀ་རེ་རེ་དུམ་བུ་རེ་ལ་བརྩི་དགོས།

- ལུང་འདྲེན་གྱི་སྔ་ཕྱི་གཉིས་སུ་ཡོད་པའི་ལུངས་ཁུངས་དང་མཇུག་བསྡུའི་ཚིག་རྣམས་ཐིག་འཕྲེང་རེ་རེར་བཞག་དགོས།

- ལུང་འདྲེན་ཚིགས་བཅད་རྣམས་ཚིགས་རྐང་རེ་རེ་བཞིན་འོག་ཏུ་ཕབ་དགོས། འོན་ཀྱང་ཚིགས་བཅད་རེ་རེ་དུམ་བུ་(block)རེ་རེར་བརྩི་བ་ལས་ཚིགས་རྐང་རེ་རེ་དུམ་བུར་བརྩི་མི་དགོས།

- དུམ་ཚན་རྣམས་ཀྱི་བར་ལ་ཐིག་འཕྲེང་སྟོང་པ་རེ་བཞག་དགོས།

- Add obsidian block IDs to each block restarting at each heading 2 or heading 3 (i.e. ^2-1-1). Don't give an ID to headings.

- བཅོས་སྒྲིག་ཟིན་པའི་ཡིག་ཆ་དེ་ .md རྣམ་ཅན་གྱི་ཡིག་ཆ་རུ་དགོས། 

- བཅོས་སྒྲིག་ཟིན་པའི་ཡིག་ཆ་འདི་ obsidian ནང་དུ་འཇོག་རྒྱུའི་ཆེ་དུ་ཡིན།
