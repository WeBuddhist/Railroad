# 般若波羅蜜多心經 (敦煌石室本)
## 1 Prologue

![[sk-iats-root-text#^1-1]]

如是我聞：一時薄伽梵住王舍城鷲峯山中，與大苾蒭眾及諸菩薩摩訶薩俱。 ^1-1

![[sk-iats-root-text#^1-2]]

爾時，世尊等入甚深明了三摩地法之異門。 ^1-2

## 2 Core Content

![[sk-iats-root-text#^2-1]]

復於爾時，觀自在菩薩摩訶薩行深般若波羅蜜多時，觀察照見五蘊體性悉皆是空。 ^2-1

![[sk-iats-root-text#^2-2]]

時，具壽舍利子，承佛威力，白聖者觀自在菩薩摩訶薩曰： ^2-2

![[sk-iats-root-text#^2-3]]

![[sk-iats-root-text#^2-4]]

「若善男子欲修行甚深般若波羅蜜多者，復當云何修學？」作是語已。觀自在菩薩摩訶薩答具壽舍利子言： ^2-3

![[sk-iats-root-text#^2-5]]

「若善男子及善女人，欲修行甚深般若波羅蜜多者，彼應如是觀察，五蘊體性皆空。」 ^2-4

![[sk-iats-root-text#^2-6]]

色即是空，空即是色。色不異空，空不異色。 ^2-5

![[sk-iats-root-text#^2-7]]

如是受、想、行、識亦復皆空。 ^2-6

![[sk-iats-root-text#^2-8]]

是故舍利子！一切法空性無相，無生無滅，無垢離垢，無減無增。 ^2-7

![[sk-iats-root-text#^2-9]]

舍利子！是故爾時空性之中，無色、無受、無想、無行亦無有識。無眼、無耳、無鼻、無舌、無身、無意。無色、無聲、無香、無味、無觸、無法。 ^2-8

![[sk-iats-root-text#^2-10]]

無眼界乃至無意識界。 ^2-9

![[sk-iats-root-text#^2-11]]

無無明亦無無明盡，乃至無老死亦無老死盡。 ^2-10

![[sk-iats-root-text#^2-12]]

無苦、集、滅、道，無智無得亦無不得。 ^2-11

![[sk-iats-root-text#^2-13]]

是故舍利子！以無所得故，諸菩薩眾依止般若波羅蜜多，心無障礙，無有恐怖，超過顛倒，究竟涅槃。 ^2-12

![[sk-iats-root-text#^2-14]]

三世一切諸佛亦皆依般若波羅蜜多故，證得無上正等菩提。 ^2-13

![[sk-iats-root-text#^2-15]]

舍利子！是故當知般若波羅蜜多大密咒者，是大明咒，是無上咒，是無等等咒。能除一切諸苦之咒，真實無倒。故知般若波羅蜜多是祕密咒。」 ^2-14

![[sk-iats-root-text#^2-16]]

即說般若波羅蜜多咒曰：「峩帝　峩帝　波囉峩帝　波囉僧峩帝　菩提　莎訶 ^2-15

## 3 Epilogue

![[sk-iats-root-text#^3-1]]

「舍利子！菩薩摩訶薩應如是修學甚深般若波羅蜜多。」 ^3-1

![[sk-iats-root-text#^3-2]]

爾時，世尊從彼定起，告聖者觀自在菩薩摩訶薩曰：「善哉，善哉！善男子！如是，如是！ ^3-2

![[sk-iats-root-text#^3-3]]

如汝所說。彼當如是修學般若波羅蜜多。一切如來亦當隨喜。」 ^3-3

![[sk-iats-root-text#^3-4]]

時薄伽梵說是語已。 ^3-4

![[sk-iats-root-text#^3-5]]

具壽舍利子，聖者觀自在菩薩摩訶薩，一切世間天、人、阿蘇羅、乾闥婆等，聞佛所說，皆大歡喜，信受奉行。 ^3-5
