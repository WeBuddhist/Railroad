## Gloss Divergences

- ⚑ **སྤྱད་པར་བྱ་ vs བསླབ་པར་བྱ་**: The source text and Commentaries 16 and 34 use 'སྤྱད་པར་བྱ་' (spyad par bya, should practice/engage). Commentaries 7, 10, 23, and 33 use 'བསླབ་པར་བྱ་' (bslab par bya, should study/train). The primary reading 'སྤྱད་པར་བྱ་' is retained as it aligns with the UCCA and is supported by multiple commentaries.

## Standardized text

དེ་ནས་བཅོམ་ལྡན་འདས་ཏིང་ངེ་འཛིན་དེ་ལས་བཞེངས་ཏེ། བྱང་ཆུབ་སེམས་དཔའ་སེམས་དཔའ་ཆེན་པོས་འཕགས་པ་སྤྱན་རས་གཟིགས་དབང་ཕྱུག་ལ་ལེགས་སོ་ཞེས་བྱ་བ་བྱིན་ནས། ལེགས་སོ་ལེགས་སོ། རིགས་ཀྱི་བུ་དེ་དེ་བཞིན་ནོ། ། རིགས་ཀྱི་བུ་དེ་དེ་བཞིན་ཏེ། ཇི་ལྟར་ཁྱོད་ཀྱིས་བསྟན་པ་བཞིན་དུ་ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པ་ཟབ་མོ་ལ་སྤྱད་པར་བྱ་སྟེ།དེ་བཞིན་གཤེགས་པ་རྣམས་ཀྱང་རྗེས་སུ་ཡི་རང་ངོ་། །

The name of the Bodhisattva was standardized from 'འཕགས་པ ་སྤྱན་རས་དབང་ཕྱུག་' to 'འཕགས་པ་སྤྱན་རས་གཟིགས་དབང་ཕྱུག་' to correct a typographical error and to include 'གཟིགས་', which is part of the standard and complete name for Avalokiteshvara, as supported by multiple commentaries (e.g., Commentary 18: 'འཕགས་པ་སྤྱན་རས་གཟིགས་ལ་'; Commentary 26: 'འཕགས་པ་སྤྱན་རས་གཟིགས་དབང་ཕྱུག་ལ་'). This correction ensures the proper identification of the deity.

## Term analysis

### `འཕགས་པ ་སྤྱན་རས་དབང་ཕྱུག་` (discrepancy)

The source text's 'འཕགས་པ ་སྤྱན་རས་དབང་ཕྱུག་' (phags pa spyan ras dbang phyug) contains a typographical error (space between 'པ' and '་') and omits 'གཟིགས་' (gzigs) from the standard name 'འཕགས་པ་སྤྱན་རས་གཟིགས་དབང་ཕྱུག་' (Noble Avalokiteshvara). Multiple commentaries (18, 19, 26, 32, 34) explicitly include 'གཟིགས་', which is crucial for the complete and standard name of the deity. This omission is a meaning-altering variant as it changes the full designation of the Bodhisattva.

### `སྤྱད་པར་བྱ་སྟེ།`

The term 'སྤྱད་པར་བྱ་སྟེ།' (spyad par bya ste, should practice/engage) is used in the source text and supported by Commentaries 16 and 34. Some other commentaries (7, 10, 23, 33) use 'བསླབ་པར་བྱ་' (bslab par bya, should study/train). While these terms have distinct nuances, the source text's reading is well-attested and semantically appropriate in the context of engaging with wisdom. This is considered a divergence in interpretation or a synonymous variant rather than a fundamental meaning-altering error requiring standardization of the text itself, especially since the source reading is supported.
