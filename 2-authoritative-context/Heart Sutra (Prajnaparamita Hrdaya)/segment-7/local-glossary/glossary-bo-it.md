## Term analysis

### `འདི་སྐད་ཅེས་ (di skad ces)`

The source text uses 'འདི་སྐད་ཅེས་' (di skad ces, spoke this way), while some commentaries (e.g., Commentary 2, 6, 8) use 'དེ་སྐད་ཅེས་' (de skad ces, spoke that way) when referring to or quoting the speech. Both 'འདི་སྐད་ཅེས་' and 'དེ་སྐད་ཅེས་' function as common ways to introduce direct speech, meaning 'thus' or 'as follows.' This is a deictic variation (proximal vs. distal demonstrative) that does not fundamentally alter the semantic meaning in this context. Therefore, it is not considered a meaning-altering textual variant.
