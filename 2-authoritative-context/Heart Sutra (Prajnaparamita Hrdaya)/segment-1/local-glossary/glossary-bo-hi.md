## Gloss Divergences

⚑ **པྲ་ཛྙ་ vs པྲ་ཛྙཱ་**: The source text reads 'པྲ་ཛྙ་' (prajnya), lacking the long 'ā' vowel, while the standard Sanskrit 'Prajñā' requires 'པྲ་ཛྙཱ་' (prajnyA).
⚑ **པ་ར་མི་ཏཱྀ་ vs པཱ་ར་མི་ཏ་**: The source text reads 'པ་ར་མི་ཏཱྀ་' (pa ra mi tI), incorrectly using a short 'a' for 'Pā' and a long 'ī' for 'tā'. The standard Sanskrit 'Pāramitā' requires 'པཱ་ར་མི་ཏ་' (pA ra mi ta).
⚑ **ཧྲད་ཡ vs ཧྲི་ད་ཡ་**: The source text reads 'ཧྲད་ཡ' (hrad ya), which is an incorrect transliteration for the Sanskrit 'Hṛdaya', which should be 'ཧྲི་ད་ཡ་' (hri da ya).

## Standardized text

༄༅། །རྒྱ་གར་སྐད་དུ། བྷ་ག་བ་ཏི་པྲ་ཛྙཱ་པཱ་ར་མི་ཏ་ཧྲི་ད་ཡ།

Multiple meaning-altering transliteration variants were corrected in the Sanskrit title. The original 'པྲ་ཛྙ་' (prajnya) was corrected to 'པྲ་ཛྙཱ་' (prajnyA) (Prajñā), 'པ་ར་མི་ཏཱྀ་' (pa ra mi tI) was corrected to 'པཱ་ར་མི་ཏ་' (pA ra mi ta) (Pāramitā), and 'ཧྲད་ཡ' (hrad ya) was corrected to 'ཧྲི་ད་ཡ་' (hri da ya) (Hṛdaya). These corrections align the Tibetan transliteration with the standard Sanskrit 'Bhagavatī Prajñāpāramitā Hṛdaya', as confirmed by multiple Sanskrit translations and Commentary 7's explicit statement of the title as 'བྷ་ག་ཝ་ཏི་པྲ་ཛྙཱ་པཱ་ར་མི་ཏ་ཧྲི་ད་ཡ་'.

## Term analysis

### `རྒྱ་གར་སྐད་དུ།`

This phrase 'In the Indian language' is standard and appears consistently across all relevant commentaries and translations. No meaning-altering variant found.

### `བྷ་ག་བ་ཏི་`

The transliteration 'བྷ་ག་བ་ཏི་' (Bhagavati) is consistent with the standard Sanskrit 'Bhagavatī'. While the final 'ī' vowel is long in Sanskrit, 'ཏི་' is a common Tibetan transliteration for 'ti' even when long. Commentary 7 also uses 'བྷ་ག་ཝ་ཏི་'. No meaning-altering variant.

### `པྲ་ཛྙ་` (discrepancy)

The source text uses 'པྲ་ཛྙ་' (prajnya), which lacks the long 'ā' vowel mark. The authoritative Sanskrit 'Prajñā' requires 'པྲ་ཛྙཱ་' (prajnyA) with the 'ཱ་' (A) vowel sign. This is a meaning-altering difference in precise transliteration, as it changes the vowel length. Commentary 7 and the Sanskrit translations confirm 'Prajñā'.

### `པ་ར་མི་ཏཱྀ་` (discrepancy)

This term contains two meaning-altering transliteration discrepancies. The first syllable 'པ་' (pa) should be 'པཱ་' (pA) to correctly transliterate the long 'ā' in 'Pāramitā'. The final syllable 'ཏཱྀ་' (tI) incorrectly uses a long 'ī' vowel. The authoritative Sanskrit 'Pāramitā' requires a long 'ā' vowel, which should be transliterated as 'ཏཱ་' (tA). This is a clear meaning-altering variant in transliteration. Commentary 7 and the Sanskrit translations confirm 'Pāramitā'.

### `ཧྲད་ཡ།` (discrepancy)

The source text uses 'ཧྲད་ཡ' (hrad ya). The authoritative Sanskrit 'Hṛdaya' is consistently transliterated as 'ཧྲི་ད་ཡ་' (hri da ya) in Tibetan, as seen in Commentary 7 and standard texts. The 'ཧྲད་' (hrad) is an incorrect transliteration for 'Hṛ'. This is a meaning-altering variant in transliteration.
