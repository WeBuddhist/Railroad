## Gloss Divergences

- ⚑ **མེད་ vs མ་ཡིན**: Commentary 3 uses 'མ་ཡིན་' (ma yin, is not) for all negations (e.g., 'མིག་མ་ཡིན' instead of 'མིག་མེད'), while the source text and most other commentaries use 'མེད་' (med, does not exist). Both convey negation, but 'མེད་' is more common for expressing non-existence in this context.

## Standardized text

No meaning-altering textual variants were found. The source text's reading, using 'མེད་' (med) for negation, is consistently supported by the majority of commentaries. While Commentary 3 uses 'མ་ཡིན་' (ma yin), this is considered a lexical variant that does not fundamentally alter the core semantic meaning of negation in the context of the Prajñāpāramitā, which emphasizes the non-existence or non-inherent existence of these phenomena.
